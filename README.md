## About:

This version include both Frontend and Backend API

## Environment:

```bash
Node version: > v12.22.0
```

## Scripts:

First, install the dependencies:

```bash
yarn
```

Run development server:

```bash
yarn dev
```

Build code:

```bash
yarn build
```

Start code:

```bash
yarn start
```

Linting:

```bash
yarn run lint
```
