## About:

This version only include Frontend code with dummy api.

## Environment:

```bash
Node version: > v12.22.0
```

## Scripts:

First, install the dependencies:

```bash
npm run install
# or
yarn
```

Run development server:

```bash
npm run dev
# or
yarn dev
```

Build code:

```bash
npm run build
# or
yarn build
```

Start code:

```bash
npm start
# or
yarn start
```

Linting:

```bash
npm run lint
# or
yarn run lint
```
